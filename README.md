# LogicGrid

A modern, responsive web-based Sudoku application designed to provide an engaging puzzle-solving experience through interactive gameplay, multiple difficulty levels, and real-time game assistance.

## Live Demo

Live Demo - https://logicgrid-sudoku.vercel.app/

## Features

### Gameplay

* Supports **4×4** and **9×9** Sudoku grids
* **3 difficulty levels**: Easy, Medium, Hard
* Dynamically generated Sudoku puzzles
* Real-time puzzle interaction

### Interactive Features

* Undo previous moves
* Hint system
* Lives-based gameplay
* In-game timer
* Best score tracking
* Pause and quit functionality
* Responsive dark-themed interface

### Algorithm

* Implemented a **recursive backtracking algorithm** to precompute valid Sudoku solutions.
* Uses the precomputed solution for efficient validation, hint generation, and gameplay management.
